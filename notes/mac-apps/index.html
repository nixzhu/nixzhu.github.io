作为 App 开发者会先安装的 Mac App

翻墙

  • ClashX，不论是作为开发者还是作为一个普通人，我们都需要完整的互联网，希望有一天我们不再需要它

开发

  • Xcode，作为 App 开发者，必须安装的工具。当然，有经验的开发者不会从 Mac App Store 下载，而是从网站下载，除了安装过程更可控，也方便多版本共存。此外，我不信任任何第三方下载方式
  • Fork，Git 客户端。虽然日常也在命令行中使用 Git（开分支或做合并），有时也直接在 Xcode 中 commit，但有个 GUI 客户端仍然比较方便（切分支或查看代码）
  • A Companion for SwiftUI，如其名字所示，可帮你探索、学习 SwiftUI，省去许多查找其用法的时间
  • Ducky，帮助开发者从 JSON 推断出多种语言的模型以节省时间，还提供许多实用的选项（由本人开发，参考 Introducing Ducky）
  • SwiftFormat for Xcode，提供一个 Xcode 的源代码编辑器扩展。在 Xcode 中设置好快捷键后，能在编写 Swift 代码时更好地格式化（参考通过快捷键在 Xcode 中使用 SwiftFormat）
  • System Color Picker，简单的颜色提取工具，支持多种输出格式，比系统自带的好用
  • Bakery，帮助开发者给 Xcode 项目准备临时的占位图标
  • Boop，帮助处理或转换文本的工具，开发者才懂
  • ControlRoom，控制器 iOS 模拟器，方便调试，例如触发 Deep Link，构造推送等。自行 Build & Run 一次，之后就可以通过 Cmd + Space 触发 Spotlight 来搜索使用它

文字处理

  • Typora，Markdown 编辑器，主要用来写文档。有所见即所得模式，也有源码模式，还有很好的快捷键支持，写起来很舒服（比如这篇文章）

社交

  • Twitter，普通人也有“鸣叫”的权力，还能帮你认识全世界各领域有趣的人
